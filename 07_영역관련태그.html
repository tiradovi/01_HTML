style 태그 내용 적용 확인

block 형식

    화면 공간 수직 분할
    (세로 화면을 가로로 잘라서 층을 나눔)

    한층을 모두 차지함

    width, height 와 같은 크기 지정 속성 사용 가능

    block 형식의 태그
    h1~h6(제목), p/pre(문단), hr(수평선), div(영역 분할), 시맨틱 태그(div 태그에 이름을 넣는 태그)

h3는 블록 형식 태그입니다.

    블록 형식은 작성된 내용이 짧더라도 한줄을 모두 차지하기에
    다음 태그내용은 다음 줄에 작성됩니다.
    
    pre옆에 가로로 이어서 작성 X
    태그를 열고 닫을 때마다 층이 구분

inline 형식

        화면을 수평 분할
        (가로 화면을 세로로 잘라서 호(열)을 나눔)

        한층을 모두 차지하지 않고 내부에 작성된 내용 크기만큼 차지

        width, height 와 같은 크기 지정 속성 사용 불가능

        inline 형식의 태그
        b(굵게), i(기울이기), mark(글 배경칠하기), span(영역 분할) 등

inline형식은 행을 분할하고 한 열에서 이어서 작성하기에 줄이 바뀌지 않고 옆으로 작성
첫 번째 span 두 번째 span 세 번째 span

inline - block 형식

    inline의 특징 중 수평분할(줄을 세로로 자름, 열 나눔)
    block의 특징 중 크기 지정 가능
    장점을 합쳐놓은 태그내용은

    대표적인 태그 : img 태그, iframe 태그

    iframe 태그: 웹 문서(html)내부에 또 다른 웹 문서를 추가하는 태그
