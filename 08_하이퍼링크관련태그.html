a(anchor : 연결, 고정) 태그

        현재 HTML 문서와 다른 HTML 문서를 연결 시켜
        페이지를 이동할 수 있게 만드는 태그

        a 태그로 감싸진 글/내용== 하이퍼 텍스트

        하이퍼 텍스트는 다른 문서와 구분되는 HTML만의 핵심기능

a 태그의 속성

        href(HyperText REFerence): 연결할 주소를 지정하는 속성
        target = "_blank" : 새 탭에서 열기
  1. 글자관련 태그로 이동
  2. 표관련 태그로 이동

        id(식별자) 속성:
            하나의 HTML 문서 내부에서
            각각의 HTML 태그를 구분하는 식별 값

        class 는 중복으로 이름 사용 가능
        id    는 중복으로 이름 사용 지양
